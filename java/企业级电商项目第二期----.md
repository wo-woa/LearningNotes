## 第2章 Lombok框架集成及原理解析

```java
@EqualsAndHashCode(of = "id")
@ToString(exclude = "id")
@ToString(exclude = {"id","name"})
@Slf4j
```

of指定使用equals时比较的字段      

exclude将tostring时的字段排除，多个使用中括号

添加@Slf4j注解之后就可以直接使用log
